### 變數與常數

變數使用`var`，常數使用`let`。
``` swift
var variable = 10
let constant = 20
```

Swift 支援 Unicode 字元。
``` swift
let π = 3.14159
let 你好 = "你好世界"
let 🐶🐮 = "dogcow"
```

### 分號

分號可有可無。要在一行內寫出多個statement用分號隔開，但會降低可讀性。

``` swift
var x = 1
var y = 2; var z = 3
```
