《Swift 學習筆記》
=======

這是一份閱讀與練習的心得，目的不在編寫完整的說明文件，只是透過文字驗證自己學會多少，順便記錄與註解在 playground 上練習。

所有練習都是在 xcode 6.4 上完成，符合 swift 1.2 的語言標準，每個範例都能複製貼上 playground 重現練習過程。

任何指教歡迎寄信到 printk@gmail.com。想進一步認識 Swift 的朋友，建議參考最下方的連結。
